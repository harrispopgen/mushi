infer_history(change_points, mu0, eta=None, eta_ref=None, mu_ref=None, infer_eta=True, infer_mu=True, alpha_tv=0, alpha_spline=0, alpha_ridge=0, beta_tv=0, beta_spline=0, beta_rank=0, hard=False, beta_ridge=0, max_iter=1000, s0=1, max_line_iter=100, gamma=0.8, tol=0, loss='prf', mask=None, verbose=False, folded=False)[source]¶

Perform sequential inference to fit \(\eta(t)\) and \(\mu(t)\)

Parameters

  • change_points (array) – epoch change points (ordered times > 0)

  • mu0 (float64) – total mutation rate (per genome per generation)

  • eta (Optional[eta]) – initial demographic history. By default, a constant MLE is computed

  • eta_ref (Optional[eta]) – reference demographic history for ridge penalty. If None, the constant MLE is used

  • mu_ref (Optional[mu]) – reference MuSH for ridge penalty. If None, the constant MLE is used

  • infer_eta (bool) – perform \(\eta\) inference if True

  • infer_mu (bool) – perform \(\mu\) inference if True

  • loss (str) – loss function, ‘prf’ for Poisson random field, ‘kl’ for Kullback-Leibler divergence, ‘lsq’ for least-squares

  • mask (Optional[array]) – array of bools, with False indicating exclusion of that frequency

  • alpha_tv (float64) – total variation penalty on \(\eta(t)\)

  • alpha_spline (float64) – L2 on first differences penalty on \(\eta(t)\)

  • alpha_ridge (float64) – L2 for strong convexity penalty on \(\eta(t)\)

  • hard (bool) – hard rank penalty on \(\mu(t)\) (non-convex)

  • beta_tv (float64) – total variation penalty on \(\mu(t)\)

  • beta_spline (float64) – penalty on \(\mu(t)\)

  • beta_rank (float64) – rank penalty on \(\mu(t)\)

  • beta_ridge (float64) – L2 penalty on \(\mu(t)\)

  • max_iter (int) – maximum number of proximal gradient steps

  • tol (float64) – relative tolerance in objective function (if 0, not used)

  • s0 (int) – max step size

  • max_line_iter – maximum number of line search steps

  • gamma (float64) – step size shrinkage rate for line search

  • verbose (bool) – print verbose messages if True

  • folded (bool) – if False, infer \(\eta(t)\) using unfolded SFS. If True, can only be used with infer_mu=False, and infer \(\eta(t)\) using folded SFS.

Return type

None

mushi.optimization.acc_prox_grad_method(x, g, grad_g, h, prox, tol=1e-10, max_iter=100, s0=1, max_line_iter=100, gamma=0.8, verbose=False)[source]¶

Nesterov accelerated proximal gradient method with backtracking line search 1.

The optimization problem solved is:

\[\min_x g(x) + h(x)\]

where \(g\) is differentiable, and the proximal operator for \(h\) is available.

Parameters

  • x (ndarray) – initial point

  • g (Callable[[ndarray], float64]) – differentiable term in objective function

  • grad_g (Callable[[ndarray], float64]) – gradient of g

  • h (Callable[[ndarray], float64]) – non-differentiable term in objective function

  • prox (Callable[[ndarray, float64], float64]) – proximal operator corresponding to h

  • tol (float64) – relative tolerance in objective function for convergence

  • max_iter (int) – maximum number of proximal gradient steps

  • s0 (float64) – initial step size

  • max_line_iter (int) – maximum number of line search steps

  • gamma (float64) – step size shrinkage rate for line search

  • verbose (bool) – print convergence messages

Return type

ndarray

Returns

solution point

mushi.optimization.three_op_prox_grad_method(x, g, grad_g, h1, prox1, h2, prox2, tol=1e-10, max_iter=100, s0=1, max_line_iter=100, gamma=0.8, ls_tol=0, verbose=False)[source]¶

Three operator splitting proximal gradient method with backtracking line search 2.

The optimization problem solved is:

\[\min_x g(x) + h_1(x) + h_2(x)\]

where \(g\) is differentiable, and the proximal operators for \(h_1\) and \(h_2\) are available.

Parameters

  • x (ndarray) – initial point

  • g (Callable[[ndarray], float64]) – differentiable term in objective function

  • grad_g (Callable[[ndarray], float64]) – gradient of g

  • h1 (Callable[[ndarray], float64]) – 1st non-differentiable term in objective function

  • h2 (Callable[[ndarray], float64]) – 2nd non-differentiable term in objective function

  • prox1 (Callable[[ndarray, float64], float64]) – proximal operator corresponding to h1

  • prox2 (Callable[[ndarray, float64], float64]) – proximal operator corresponding to h2

  • tol (float64) – relative tolerance in objective function for convergence

  • max_iter (int) – maximum number of proximal gradient steps

  • s0 (float64) – step size

  • max_line_iter (int) – maximum number of line search steps

  • gamma (float64) – step size shrinkage rate for line search

  • ls_tol (float64) – line search tolerance

  • verbose (bool) – print convergence messages

Return type

ndarray

Returns

solution point

References

2

Pedregosa, Gidel, Adaptive Three Operator Splitting in Proceedings of the 35th International Conference on Machine Learning, Proceedings of Machine Learning Research., J. Dy, A. Krause, Eds. (PMLR, 2018), pp. 4085–4094.
