mushi.histories.mu¶

class mu(change_points: numpy.array, vals: numpy.ndarray, mutation_types: List[str])[source]¶

Bases: mushi.histories.History

Mutation spectrum history (MuSH)

Parameters

  • change_points (array) – epoch change points (times)

  • Z – matrix of constant values for each epoch (rows) in each mutation type (columns)

  • mutation_types (List[str]) – list of mutation type names (optional None element)

Example

>>> import mushi
>>> import numpy as np

>>> mu = mushi.mu(np.array([1, 10, 100, 1000]),
...               np.ones((5, 2)),
...               ['TCC>TTC', 'GAA>GGA'])
