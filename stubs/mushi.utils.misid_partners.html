mushi.utils.misid_partners

misid_partners(mutation_types)[source]

Ancestral state misidentification partner indices. Mutation type None is self-partnered.

Parameters:

mutation_types (List[str]) – list of mutation type strings, e.g. [AAA>ACA, …]

Return type:

List[int]

Examples

>>> from mushi import utils

>>> utils.misid_partners(['TCC>TTC', 'GAA>GGA', None])
[1, 0, 2]
