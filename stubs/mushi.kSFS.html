class kSFS(X=None, mutation_types=None, file=None, n=None)[source]ÔÉÅ

Bases: object

Primary class for working with SFS data to infer demography \(\eta(t)\equiv 2N(t)\), or with \(k\)-SFS data to infer demography and mutation spectrum history \(\boldsymbol\mu(t)\).

Parameters:

  • ksfs_file ‚Äì path to \(k\)-SFS file, as ouput by mutyper ksfs

  • X (Array) ‚Äì \(k\)-SFS matrix

  • mutation_types (List[str]) ‚Äì list of names of X columns

  • n (int) ‚Äì number of sampled haplotypes

Examples

>>> import mushi
>>> import numpy as np

Three constructors:

  1. ksfs_file: path to k-SFS file, as ouput by mutyper ksfs

>>> ksfs = mushi.kSFS(file='ksfs.tsv')

  1. X and mutation_types (the latter may be ommitted if X is a 1D SFS array)

>>> sfs = mushi.kSFS(X=np.array([10, 5, 3, 1]))

>>> ksfs = mushi.kSFS(X=np.ones((10, 4)),
...                   mutation_types=['AAA>ACA', 'ACA>AAA',
...                                   'TCC>TTC', 'GAA>GGA'])

  1. n: number of haplotypes to initialize for simulation

>>> ksfs = mushi.kSFS(n=100)

Simulate a \(k\)-SFS under the Poisson random field model (no linkage), assign to X attribute.

Parameters:

  • eta (eta) ‚Äì demographic history

  • mu (Union[mu, float64]) ‚Äì mutation spectrum history (or constant mutation rate)

  • r (float64) ‚Äì ancestral state misidentification rate (default 0)

  • seed (int) ‚Äì random seed

Return type:

Examples

Define sample size:

>>> ksfs = mushi.kSFS(n=10)

Define demographic history and MuSH:

>>> eta = mushi.eta(np.array([1, 100, 10000]), np.array([1e4, 1e4, 1e2, 1e4]))
>>> mush = mushi.mu(eta.change_points, np.ones((4, 4)),
...                 ['AAA>ACA', 'ACA>AAA', 'TCC>TTC', 'GAA>GGA'])

Define ancestral misidentification rate:

>>> r = 0.03

Set random seed:

>>> seed = 0

Run simulation and print simulated \(k\)-SFS

>>> ksfs.simulate(eta, mush, r, seed)

infer_eta(mu0, *trend_penalty, ridge_penalty=0, folded=False, pts=100, ta=None, log_transform=True, eta=None, eta_ref=None, loss='prf', max_iter=100, tol=0, line_search_kwargs={}, trend_kwargs={}, verbose=False)[source]ÔÉÅ

Infer demographic history \(\eta(t)\)

Parameters:

  • mu0 (float64) ‚Äì total mutation rate (per genome per generation)

  • trend_penalty (Tuple[int, float64]) ‚Äì tuple (k, Œª) for \(k\)-th order trend penalty (can pass multiple for mixed trends)

  • ridge_penalty (float64) ‚Äì ridge penalty

  • folded (bool) ‚Äì if False, infer \(\eta(t)\) using unfolded SFS. If True, can only be used with infer_mu=False, and infer \(\eta(t)\) using folded SFS.

  • pts (float64) ‚Äì number of points for time discretization

  • ta (float64) ‚Äì time (in WF generations ago) of oldest change point in time discretization. If None, set automatically based on 10 * E[TMRCA] under MLE constant demography

  • log_transform (bool) ‚Äì fit \(\log\eta(t)\)

  • eta (eta) ‚Äì initial demographic history. By default, a constant MLE is computed

  • eta_ref (eta) ‚Äì reference demographic history for ridge penalty. If None, the constant MLE is used

  • loss (str) ‚Äì loss function name from loss_functions module

  • max_iter (int) ‚Äì maximum number of optimization steps

  • tol (float64) ‚Äì relative tolerance in objective function (if 0, not used)

  • line_search_kwargs (Dict) ‚Äì line search keyword arguments, see mushi.optimization.LineSearcher()

  • trend_kwargs (Dict) ‚Äì keyword arguments for trend filtering, see mushi.optimization.TrendFilter.run()

  • verbose (bool) ‚Äì print verbose messages if True

Return type:

None

Examples

Suppose ksfs is a kSFS object. Then the following fits a demographic history with 0-th order (piecewise constant) trend penalization of strength 100, assuming a mutation rate of 10 mutations per genome per generation.

>>> mu0 = 1
>>> ksfs.infer_eta(mu0, (0, 1e2))

Alternatively, a mixed trend solution, with constant and cubic pieces, is fit with

>>> ksfs.infer_eta(mu0, (0, 1e2), (3, 1e1))

The attribute ksfs.eta is now set and accessable for plotting (see eta).

Infer mutation spectrum history \(\mu(t)\)

Parameters:

  • ridge_penalty (float64) ‚Äì ridge penalty

  • rank_penalty (float64) ‚Äì rank penalty

  • hard (bool) ‚Äì hard rank penalty (non-convex)

  • mu_ref (mu) ‚Äì reference MuSH for ridge penalty. If None, the constant MLE is used

  • misid_penalty (float64) ‚Äì ridge parameter to shrink misid rates to aggregate rate

  • loss (str) ‚Äì loss function from loss_functions module

  • max_iter (int) ‚Äì maximum number of optimization steps

  • tol (float64) ‚Äì relative tolerance in objective function (if 0, not used)

  • line_search_kwargs (Dict) ‚Äì line search keyword arguments, see mushi.optimization.LineSearcher

  • trend_kwargs (Dict) ‚Äì keyword arguments for trend filtering, see mushi.optimization.TrendFilter.run()

  • verbose (bool) ‚Äì print verbose messages if True

Return type:

None

Examples

Suppose ksfs is a kSFS object, and the demography has already been fit with infer_eta. The following fits a mutation spectrum history with 0-th order (piecewise constant) trend penalization of strength 100.

>>> ksfs.infer_mush((0, 1e2))

Alternatively, a mixed trend solution, with constant and cubic pieces, and with rank penalization 100, is fit with

>>> ksfs.infer_mush((0, 1e2), (3, 1e1), rank_penalty=1e2)

The attribute ksfs.mu is now set and accessable for plotting (see mu).
